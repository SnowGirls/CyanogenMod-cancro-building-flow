# 3. Install ccusage to view cost

https://ccusage.com/guide/statusline
Add this to your ~/.claude/settings.json or ~/.config/claude/settings.json:
{
  "statusLine": {
    "type": "command",

  }

# 6. 配置读当前目录下的文件时是无需自己授权, 注意替换 __To_Your_Project_Directory__ 为真实路径
